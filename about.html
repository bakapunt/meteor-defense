About

The game

Meteor Defense is a fun, retro-styled game written using the HTML5 canvas and JavaScript. Its levels get increasingly difficult as you progress, and the user's high score is stored, encouraging users to beat their own score. The user can use either the mouse or the left/right keys to control the game. Although this website is responsive, the game doesn't yet work on mobile devices, but this may be something that I implement in the future.

External libraries

The game makes use of the JavaScript library jQuery to make it easier to access and manipulate DOM elements and handle events.

It may also be worth mentioning that I used the Bootstrap framework to help build the website's layout and to keep it consistent with the rest of my website, as well as making the website responsive.
